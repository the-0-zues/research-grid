# Synchronous Generator Sequence Reactance Parameters for 1-3 MW Gas-Fired DER Fault Studies

## Executive summary

The strongest exact match found is a Caterpillar 3520 natural-gas generator data sheet for a 1.6 MW, 2.0 MVA, 480 V, 60 Hz generator using an SR4B frame 868 alternator; it reports all five requested quantities on the machine base: \(X_d = 1.9149\), \(X'_d = 0.2500\), \(X''_d = 0.1563\), \(X_2 = 0.1571\), and \(X_0 = 0.0538\) p.u. ([Caterpillar generator data](https://www.reactpower.com/wp-content/uploads/2022/04/1586422-Generator-Data.pdf)). This row is the best directly usable parameter set for an OpenDSS fault study of a 1-3 MW natural-gas reciprocating-engine DER, provided the voltage and MVA base are either matched or converted.

Manufacturer generator-set spec sheets for Caterpillar, Cummins, Generac, Kohler, and MTU often publish power, voltage, transient performance, and alternator options but omit the full sequence reactance set. The best secondary evidence therefore comes from alternator data sheets and manufacturer engineering guides: Cummins provides typical p.u. ranges for 40-2000 kW generators and a detailed 2.5 MW/480 V alternator example, Stamford/AvK publishes a 2.0 MVA alternator with saturated and unsaturated sequence reactances, Leroy-Somer publishes 1.25-3.4 MVA low-voltage alternator family reactance tables, and Marathon publishes exact 1.143 MVA and 4.188 MVA alternator data sheets ([Cummins reactance guide](https://mart.cummins.com/imagelibrary/data/assetfiles/0056604.pdf), [Cummins 2.5 MW example](https://mart.cummins.com/imagelibrary/data/assetfiles/0056598.pdf), [Stamford AvK DSG 99 L2/10](https://www.stamford-avk.com/sites/stamfordavk/files/dsg099l2_10_50_690.pdf), [Leroy-Somer LSA 50.2](https://www.leroy-somer.com/documentation_pdf/4098_en.pdf), [Leroy-Somer LSA 52.3](https://www.leroy-somer.com/documentation_pdf/5199_en.pdf), [Marathon 743FSS4232](https://www.larmarindustries.com/wp-content/uploads/2023/08/4232_Alternator_Data_Sheet.pdf), [Marathon 1030FDL1110](https://www.marathongenerators.com/generators/docs/perf/1030FDL1110_MPR_60Hz_480V_3Ph.pdf)).

For OpenDSS specifically, the built-in Generator object does not directly consume all five quantities in the way a full sequence-network short-circuit program might. The OpenDSS Generator Dynamics/FaultStudy model uses a Thevenin equivalent with Xdp in the positive-sequence network and Xdpp in the negative-sequence network, while zero sequence is better represented with a grounded-wye/delta transformer whose impedance is set to the generator zero-sequence impedance ([OpenDSS Generator Dynamics Model](https://opendss.epri.com/GeneratorDynamicsModel.html)). Roger Dugan’s OpenDSS guidance clarifies that Xdp is the transient reactance \(X'_d\), Xdpp is normally \(X''_d\) but is used as the negative-sequence impedance, and \(X''_d\) and \(X_{neg}\) are usually similar ([OpenDSS SourceForge discussion](https://sourceforge.net/p/electricdss/discussion/861976/thread/3872111b92/)).

## OpenDSS parameter mapping and modeling implications

| Required physical parameter | Physical meaning | OpenDSS handling for Generator object |
| --- | --- | --- |
| \(X_d\) | Direct-axis synchronous reactance for the steady-state period of generator fault current. The IEEE PSRC tutorial states that \(X_d\) determines the steady-state period fault current magnitude ([IEEE PSRC tutorial](https://www.pes-psrc.org/kb/report/026.pdf)). | Not generally used by the built-in Generator object for OpenDSS power-flow initialization; keep it for documentation, external validation, or custom models. |
| \(X'_d\) | Direct-axis transient reactance. OpenDSS developer guidance states that Xdp is used to set the voltage behind \(X'_d\) for short-circuit and dynamics analysis ([OpenDSS SourceForge discussion](https://sourceforge.net/p/electricdss/discussion/861976/thread/3872111b92/)). | Use as `Xdp`, after base conversion if needed. |
| \(X''_d\) | Direct-axis subtransient reactance for the first few cycles. The IEEE PSRC tutorial states that \(X''_d\) is used to calculate the highest initial short-circuit current and gives a typical major-unit range of 0.12-0.22 p.u. ([IEEE PSRC tutorial](https://www.pes-psrc.org/kb/report/026.pdf)). | Do not blindly map to `Xdpp` if actual \(X_2\) is known; OpenDSS uses `Xdpp` as the negative-sequence impedance in the dynamics/fault model ([OpenDSS Dynamics Mode](https://opendss.epri.com/OpenDSSDynamicsMode.html)). |
| \(X_2\) | Negative-sequence reactance for unbalanced faults. The IEEE PSRC tutorial gives \(X_2 = X''_d\) for round-rotor machines and \(X_2 = (X''_d + X''_q)/2\) for salient-pole machines ([IEEE PSRC tutorial](https://www.pes-psrc.org/kb/report/026.pdf)). | Prefer actual \(X_2\) as `Xdpp`; if \(X_2\) is absent, \(X''_d\) or \((X''_d + X''_q)/2\) is a documented approximation. |
| \(X_0\) | Zero-sequence reactance for grounded-neutral line-to-ground fault paths. NATF notes that zero-sequence reactance applies only for line-to-ground faults and only when the generator neutral is grounded ([NATF generator specifications reference](https://www.natf.net/docs/natfnetlibraries/documents/resources/planning-and-modeling/natf-reference-document-generator-specifications-open.pdf)). | Use a same-kVA grounded-wye/delta transformer in front of the generator, set transformer impedance to \(X_0\), and reduce `Xdp`/`Xdpp` by that transformer impedance if following OpenDSS developer guidance ([OpenDSS Generator Dynamics Model](https://opendss.epri.com/GeneratorDynamicsModel.html)). |

## Sanity check against expected ranges

The user-provided sanity ranges of \(X_d = 0.80-1.50\), \(X'_d = 0.15-0.35\), \(X''_d = 0.10-0.25\), \(X_2 = 0.10-0.25\), and \(X_0 = 0.03-0.15\) are broadly consistent for transient, subtransient, and negative-sequence quantities, but many low-voltage alternator data sheets report \(X_d\) materially above 1.50 p.u. Cummins’ own engineering guide gives typical values for 40-2000 kW generators of \(X''_d = 0.09-0.17\), \(X'_d = 0.13-0.20\), \(X_d = 1.7-3.3\), \(X_0 = 0.06-0.09\), and \(X_2 = 0.10-0.22\), which explains why the Cat, Marathon, Generac, and Leroy-Somer \(X_d\) values should not be discarded solely for exceeding 1.50 p.u. ([Cummins reactance guide](https://mart.cummins.com/imagelibrary/data/assetfiles/0056604.pdf)).

The clearest outliers are zero-sequence values below 0.03 p.u. in the Cummins example, Marathon rows, Leroy-Somer rows, and Generac K0820124Y27 row. These low \(X_0\) values are not automatically wrong because zero-sequence reactance is strongly winding- and grounding-dependent, and NATF explicitly notes that zero-sequence reactance matters only when the generator has a grounded neutral ([NATF generator specifications reference](https://www.natf.net/docs/natfnetlibraries/documents/resources/planning-and-modeling/natf-reference-document-generator-specifications-open.pdf)). For OpenDSS SLG simulations, the interconnection transformer may dominate or block the zero-sequence path, so the practical modeling decision is whether the DER is connected through grounded-wye, delta, or grounded-wye/delta equipment rather than simply which machine \(X_0\) value is published.

## Best parameter set for an EPRI Ckt7 OpenDSS fault-classification study

If a 1.6 MW natural-gas reciprocating DER is acceptable for the study, the Caterpillar 3520/SR4B data sheet should be used as the primary machine-base set: \(MVA = 2.0\), \(kV = 0.48\), \(X_d = 1.9149\), \(X'_d = 0.2500\), \(X''_d = 0.1563\), \(X_2 = 0.1571\), and \(X_0 = 0.0538\) ([Caterpillar generator data](https://www.reactpower.com/wp-content/uploads/2022/04/1586422-Generator-Data.pdf)). For OpenDSS, set `Xdp=0.2500` and use `Xdpp=0.1571` if interpreting `Xdpp` as the negative-sequence impedance; using `Xdpp=0.1563` instead is numerically almost identical for this machine because \(X_2\) and \(X''_d\) differ by only 0.0008 p.u. ([Caterpillar generator data](https://www.reactpower.com/wp-content/uploads/2022/04/1586422-Generator-Data.pdf), [OpenDSS SourceForge discussion](https://sourceforge.net/p/electricdss/discussion/861976/thread/3872111b92/)).

For SLG faults, do not assume that the machine \(X_0\) alone sets feeder zero-sequence injection. The OpenDSS recommended practice is to model the generator as delta-connected and add a grounded-wye/delta transformer with the same kVA rating when zero sequence must be included, with transformer impedance set to the machine \(X_0\) and `Xdp`/`Xdpp` reduced by the transformer impedance if following the documented convergence-friendly approach ([OpenDSS Generator Dynamics Model](https://opendss.epri.com/GeneratorDynamicsModel.html)). If the actual DER interconnection transformer blocks zero-sequence current from the machine side, then machine \(X_0\) should not be allowed to inject zero-sequence current into the feeder, even though the manufacturer data sheet reports a nonzero \(X_0\).

## Source credibility and unresolved gaps

The most credible source for all five requested parameters is the exact Caterpillar natural-gas generator data sheet because it provides a gas-fired unit, target power class, voltage, MVA base, \(X_d\), \(X'_d\), \(X''_d\), \(X_2\), and \(X_0\) in one manufacturer document ([Caterpillar generator data](https://www.reactpower.com/wp-content/uploads/2022/04/1586422-Generator-Data.pdf)). The next most credible sources for parameter magnitudes are alternator manufacturer data sheets from Stamford/AvK, Leroy-Somer, Marathon, and Generac because reactances are alternator properties, but these sources do not prove the alternator was installed behind a natural-gas engine or turbine in a specific DER package ([Stamford AvK DSG 99 L2/10](https://www.stamford-avk.com/sites/stamfordavk/files/dsg099l2_10_50_690.pdf), [Leroy-Somer LSA 50.2](https://www.leroy-somer.com/documentation_pdf/4098_en.pdf), [Leroy-Somer LSA 52.3](https://www.leroy-somer.com/documentation_pdf/5199_en.pdf), [Marathon 743FSS4232](https://www.larmarindustries.com/wp-content/uploads/2023/08/4232_Alternator_Data_Sheet.pdf), [Generac K0820124Y27 alternator data](https://legacy.genconnect.generac.com/Media/vwDoc.axd?d=99610d97-d652-440e-8316-49fe68ccac68)).

The main unresolved gap is that common gas-generator package spec sheets often identify the alternator family but do not publish the full \(X_d\), \(X'_d\), \(X''_d\), \(X_2\), and \(X_0\) set. Generac SG1000 is an example: the spec sheet confirms a 1.0 MW natural-gas generator set and standard alternator model but does not provide the reactance set needed for sequence-network fault simulation ([Generac SG1000 spec](https://legacy.genconnect.generac.com/Media/vwDoc.axd?d=29eb8834-166f-4228-8059-d90dc01390b0)). Before finalizing the OpenDSS training dataset, the DER interconnection transformer winding, neutral grounding method, and whether the machine is connected low-voltage directly or through a step-up transformer should be documented, because those details decide whether \(X_0\) affects Ckt7 SLG fault current at all.
